Informationen

Auf dieser Webseite finden Sie interaktive Theorie zum Thema Suchalgorithmen. Sie ist so aufgebaut, dass Sie selbstständig und im eigenen Tempo aneignen werden kann.
Die einfacheren grünen Inhalte erledigen und überprüfen Sie selbstständig.
Die Lösungen der blauen Zusatzinhalte werden jeweils in der Klasse besprochen.

Wenn Sie eine Aufgabe beginnen, stellen Sie sicher, dass Sie zuerst auf den schwarzen "Beispiel laden" Knopf drücken, damit die relevanten Inhalte zur Aufgabe geladen werden. Sie können bei jeder Aufgabe prüfen, ob Sie richtig lagen, indem sie auf den Knopf "Lösung laden" drücken. Beachten Sie, dass nicht alle Lösungen eindeutig sind, es gibt also oft viele Lösungen, welche richtig sein können.
